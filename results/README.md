# Verification Results

This directory contains representative TLC outputs corresponding to the verification campaigns reported in the accompanying paper.

| Log | Description |
|------|-------------|
| `safety.log` | Complete safety verification (2 runners, adversarial model). |
| `liveness_1runner.log` | Single-runner liveness verification with fairness. |
| `liveness_without_adv.log` | Two-runner liveness verification without adversarial transitions. |
| `liveness.log` | Complete adversarial liveness verification with fairness. |
