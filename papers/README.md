# Scientific Publications

This directory contains scientific publications accompanying the Secure Runner formal specification.

## Extended Manuscript

**Formal Modelling of an Ephemeral Trust for Secure CI/CD Runners**

This manuscript is the extended version of the paper accepted at **INForum 2026**.

It includes material omitted from the conference paper due to space constraints, including:

- complete architectural description;
- formal TLA+ specification;
- authorization semantics;
- verification methodology;
- evaluation methodology;
- reproducibility information.

The manuscript is temporarily hosted in this repository until it becomes available as an arXiv preprint.
